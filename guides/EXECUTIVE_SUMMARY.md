## THE SCORING SYSTEMS EXPLAINED

### Buying Climax (BC) Score: 0-28 Points

Each of 7 signals worth 0-4 points:
```
0-4 points:  One signal triggered (minimal risk)
5-12 points: Two signals triggered (monitor phase)
13-20 points: Three-four signals triggered (review phase - caution)
21-28 points: Five+ signals triggered (action required)
```

| BC Score | Meaning | Action |
|----------|---------|--------|
| 0-8 | No climax risk | MONITOR - continue position |
| 9-17 | Mixed signals | NEUTRAL - watch for confirmation |
| 18-23 | Caution warranted | REVIEW - prepare exit plan |
| 24-28 | Climax likely | SCRUTINIZE - exit soon |

### Spring Pattern Score: 0-12 Points

Each component worth 0-2 points:
```
0-4 points:  Weak setup, don't enter
5-8 points:  Possible setup, needs confirmation
9-12 points: Strong setup, entry consideration
```

| Spring Score | Meaning | Action |
|--------------|---------|--------|
| 0-4 | No spring pattern | SKIP - find better entry |
| 5-8 | Possible pattern | WAIT - for more confirmation |
| 9-12 | Strong pattern | ENTER - good risk/reward |

## COMMON QUESTIONS

**Q: Do I HAVE to use both tiers?**
A: No. Use just Tier 1 if you want daily screening only. Add Tier 2 only when you're making actual trade decisions. Most users: Tier 1 daily ($0.001) + Tier 2 on top 2-3 tickers ($0.05) = $0.10/week.

**Q: Can I use just Claude Haiku for everything?**
A: Haiku gives you scores; it doesn't make decisions. For "Should I exit now?" questions, Sonnet's deeper analysis is worth the $0.025 cost. Think of it as trade insurance.

**Q: What if I'm on a budget?**
A: Use Haiku-only for screening. When you see BC score 24-28, that's signal enough to start unwinding manually. You don't need Sonnet confirmation for extreme scores.

**Q: How accurate is the BC scoring?**
A: Haiku scores match Sonnet scores 95%+ of the time. Where they differ, it's usually ±2 points (not enough to change action). For stakes where that matters, use Tier 2.

**Q: Can I use this for crypto, forex, commodities?**
A: Yes. The Wyckoff signals are universal to liquid markets. Just ensure your data source covers the asset (Polygon covers US stocks/options; you'd need different API for crypto).

**Q: What's the best starting point if I'm new to this?**
A: 
1. Read BC Master Guide (understand signals)
2. Try Copy/Paste Template 1 (single ticker)
3. Try Copy/Paste Template 2 (batch 5 tickers)
